Resumo do Projeto de Landing Page – Banco Jota (One Page)
Objetivo:
Landing page one page para captação e qualificação de leads de consórcio consultivo do Banco Jota, com formulário integrado ao WhatsApp e IA de atendimento. Design profissional e copywriting de conversão.

Stack de Desenvolvimento
React

JavaScript

Tailwind CSS

Next.js

Landing page moderna, responsiva e mobile-first, usando Next.js e Tailwind para estilização customizada conforme a identidade visual da logo enviada.

Requisitos e Especificações
One Page Responsiva:
Layout único, 100% responsivo, mobile-first e navegação fluida.

Menu de Navegação (Nav):
Links internos por âncora para rolagem suave até as seções: Início, Sobre, Benefícios, Como Funciona, Depoimentos, Fale Conosco.

Header de Impacto:
Logo Banco Jota, título forte e CTA. Destaque para atendimento humano/consultivo.

Seção de Vídeo ou Imagem:
Espaço para vídeo institucional ou imagem destaque.

Sobre:
Descrição clara do propósito, diferenciais e consultoria do Banco Jota.

Benefícios:
Lista visual de benefícios (icons, badges ou cards).

Como Funciona:
Passo a passo simples:

Preenchimento do formulário

Contato via WhatsApp

Qualificação consultiva

Proposta personalizada

Acompanhamento até contemplação e pós-venda

Depoimentos / Prova Social:
Espaço para depoimentos de clientes reais/fictícios.

Formulário de Lead:
Integração direta ao WhatsApp (ou automação IA), campos essenciais, consentimento LGPD (checkbox), redirecionamento para página de “Obrigado”.

CTAs:
Botões visíveis, cores do branding, textos como “Fale com um Consultor”, “Simule seu Consórcio”.

Rodapé Completo:
Links para Política de Privacidade, Termos de Uso, LGPD, redes sociais, WhatsApp, endereço, e-mail.

Páginas Legais e LGPD:

Política de Privacidade

Termos de Uso

LGPD
Checkbox de consentimento obrigatório no formulário.

Design e Performance:
Visual moderno, elegante e alinhado à marca (usar cores sugeridas).
Animações suaves (hover, fade-in, botões).
SEO otimizado, meta tags, sitemap, estrutura amigável.
Imagens otimizadas (Next/Image).
README básico para futuras edições.

Funcionalidades Extras:

FAQ (opcional)

Área para baixar e-books ou materiais

reCAPTCHA ou similar contra spam

Popup/cookie banner LGPD

README/documentação simples

Prompt para Gerador ou Desenvolvedor
Crie uma landing page one page para captação de leads de consórcio do Banco Jota, utilizando React, JavaScript, Tailwind CSS (com a paleta de cores baseada na logo fornecida) e Next.js. O projeto deve ser responsivo, moderno, mobile-first, com navegação por âncoras (scroll suave), formulário integrado ao WhatsApp (ou IA), header de impacto, seções de sobre, benefícios, como funciona, depoimentos, CTA, rodapé completo e páginas de Política de Privacidade, Termos de Uso e LGPD. O formulário deve conter consentimento LGPD obrigatório. Otimize para SEO, performance e copywriting de conversão. O design deve ser visualmente alinhado à marca, usando as cores sugeridas e imagens otimizadas. Inclua README de instrução. Menu nav deve apontar para blocos internos da página. Caso possível, inclua seção de FAQ, área para download de materiais e proteção anti-spam.
