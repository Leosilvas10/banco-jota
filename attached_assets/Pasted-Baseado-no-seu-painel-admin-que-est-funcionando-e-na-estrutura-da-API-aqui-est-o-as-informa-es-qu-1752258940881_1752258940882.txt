Como Integrar
Para buscar dados existentes:

Fazer GET para /api/content/landing/banco-jota
Usar os dados retornados para preencher o editor
Para salvar alterações:

Fazer PUT para /api/content/landing/banco-jota
Enviar o objeto JSON completo com as modificações
Tratamento de Erros:

Status 200: Sucesso
Status 400: Dados inválidos
Status 404: Conteúdo não encontrado
Status 500: Erro interno
